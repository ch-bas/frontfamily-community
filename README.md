## What is FrontFamily?

FrontFamily is a free tool for converting UI component code between frontend frameworks and comparing them side-by-side.

- **Framework Converter** — paste code from MUI, Chakra UI, Ant Design, EUI, Mantine, Bootstrap, or Tailwind, get the equivalent in another framework instantly
- **CLI** — `npx @frontfamily/cli eject auth-form -f react-mui` — 162 production-ready templates across 9 frameworks
- **Migration Guides** — 9 in-depth articles with 219 searchable component mappings
- **Side-by-Side Showroom** — compare up to 3 frameworks rendering the same pattern with real DOM metrics

## How to contribute

We welcome contributions in these forms:

### 1. Report a bug

Something broken in the converter, CLI, or website? [Open a bug report](../../issues/new?template=bug_report.md).

### 2. Request a feature

Want a new framework, pattern, or conversion path? [Open a feature request](../../issues/new?template=feature_request.md).

### 3. Fix a component mapping

Found a prop conversion that's wrong? An unmapped component? [Submit a mapping correction](../../issues/new?template=mapping_correction.md).

### 4. Suggest a migration guide

Have a framework migration path that isn't covered? [Suggest a guide](../../issues/new?template=guide_suggestion.md).
